A **modern, responsive React application** that provides **real-time weather information** and a **5-day forecast** using the [OpenWeather API](https://openweathermap.org/api). This project demonstrates **API integration, responsive UI, geolocation, and React state management**, along with professional design and interactivity.

---

## **Features**

- Automatically detects your **current location** and displays weather data.
- Search for **any city** to view current weather and 5-day forecast.
- Displays **temperature, humidity, wind speed, and weather conditions**.
- Responsive design with **blurred cards, gradient backgrounds, and clean typography**.
- Horizontal scroll for forecast cards on mobile devices for a professional look.
- Uses **OpenWeather icons** for visual representation of weather conditions.

---

## **Technologies Used**

- **Frontend:** React, JavaScript, HTML, CSS  
- **API:** OpenWeather API (current weather + 5-day forecast)  
- **Design:** CSS Flexbox & Grid, gradient backgrounds, card layout, responsive design
